# NCKU-lab-Bikes-analysis

## 目標網頁
1. https://mtbdatabase.com/bikes/
2. https://roadbikedatabase.com/

## 呈現結果

* bike_spec.xlsx
  *  year 年份
  *  brand 腳踏車型號
  *  name 腳踏車型號
  *  price 價格
  *  Size 尺寸
  *  Weight 重量
  *  Colour 顏色
  *  specification 規格 -> 依照主分類、子分類列出
  *  link 網站連結
  *  source 資料庫來源
